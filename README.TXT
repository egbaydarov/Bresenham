--STARTUP GUIDE
Make "Drag and drop" action with mouse to draw a figure

Press number on the "top" alphanumeric keyboard to switch between tools.

"1": Line drawing tool
"2": Ellipsis drawing tool
"3": Circle drawing tool
"F": Enable/Disable fill mode

"8": Line clipping tool (Draw lines before use it)
"9": ScanLine fill tool
"0": Flood fill tool
"H": Switch drawing/help (Clear)
"Esc": Exit
