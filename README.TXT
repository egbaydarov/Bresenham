--STARTUP GUIDE
Make "Drag and drop" action with mouse to draw a figure

Press number on the "top" alphanumeric keyboard to switch between scenes.

"1": Line drawing scene
"2": Circle drawing scene
"3": Ellipsis drawing scene
"0": This page
"Esc": Exit
