--STARTUP GUIDE
Click on canvas to add a point.

Make "Drag and drop" action with mouse to move points if possible

Press number on the "top" alphanumeric keyboard to switch between tools.

"1": Bezier for N
"2": Bezier by order 3
"3": Complex Bezier by order 3
"4": B Spline
"5": Complex B spline
"5": De Boor spline

"L": Loop/Unloop curve
"C": Remove all points
"A": Show alternative curve

"H": Switch drawing/help
"Esc": Exit
